## 2. Business Case

### 2.1 Problem Statement
Traditional Digital-Out-of-Home (DOOH) advertising suffers from:
- Lack of real-time control and updates
- Poor targeting capabilities
- High entry barriers for small businesses
- Fragmented reporting and analytics
- Static content that cannot adapt to time or location

### 2.2 Opportunity
AdMotion transforms urban vehicles into dynamic advertising platforms, providing:
- Real-time content updates and control
- AI-driven campaign optimization
- Affordable access for SMEs
- Comprehensive analytics and reporting
- Location-based and time-sensitive advertising

### 2.3 Expected Benefits
- Revenue generation for fleet owners
- Cost-effective advertising for local businesses
- Data-driven insights for advertisers
- Scalable platform for future expansion

## 3. Project Objectives

### 3.1 Primary Objectives
1. Develop a comprehensive admin dashboard for fleet and campaign management
2. Create a vehicle display application for real-time ad playback
3. Implement AI-powered ad scheduling and optimization
4. Ensure secure, scalable, and maintainable system architecture

### 3.2 Success Criteria
- Successful registration and management of 100+ vehicles
- Real-time ad synchronization across all active vehicles
- AI algorithm achieving 95% efficiency in ad distribution
- System uptime of 99.9% during operational hours

## 4. Project Scope

### 4.1 In Scope
- Admin dashboard with user management, vehicle registration, ad management
- Vehicle display app with real-time sync and offline capabilities
- AI scheduling system for automated ad distribution
- Firebase backend with real-time database
- Security features including authentication, authorization, and data protection
- Responsive web interface and PWA capabilities

### 4.2 Out of Scope
- Hardware development for LED displays
- Physical installation of displays on vehicles
- Third-party payment processing integration
- Mobile applications for end-users
